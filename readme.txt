# 🍕 Pizza Sales Analysis Project

## 📌 Overview
This project focuses on analyzing pizza sales data to extract meaningful insights and build an interactive dashboard.  
The goal is to understand sales patterns, customer behavior, and business performance using **SQL and Excel**.

---

## 🎯 Objectives
- Clean and transform raw data  
- Analyze sales trends and patterns  
- Identify top-performing products  
- Understand customer ordering behavior  
- Build a dashboard for visualization  

---

## 🛠️ Tools & Technologies
- SQL (MySQL)
- Microsoft Excel

---

## 📂 Project Workflow

### 1️⃣ Data Cleaning
- Handled missing/null values  
- Formatted date and time columns  
- Created new calculated columns  

### 2️⃣ Data Analysis (SQL)
- Used aggregate functions (SUM, COUNT, AVG)  
- Analyzed:
  - Total revenue  
  - Orders per day/hour  
  - Sales by category & size  
- Used GROUP BY, JOIN, and subqueries  

### 3️⃣ Data Visualization (Excel)
- Built an interactive dashboard  
- Used:
  - Pivot Tables  
  - Charts (Bar, Pie, Line)  
  - Slicers for filtering  

---

## 📊 Key Insights
- Peak sales occur during specific hours of the day  
- Certain pizza categories generate higher revenue  
- Large size pizzas contribute significantly to total sales  
- Weekends show higher order volume compared to weekdays
